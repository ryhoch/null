No servers  ·  No accounts  ·  No metadata

→ End-to-end encrypted via ECDH + AES-256-GCM
→ Peer-to-peer WebRTC — messages never touch a server
→ Identity is your Ethereum wallet address
→ Offline queue — messages deliver when recipient reconnects
→ Open source · self-hostable signaling relay
 
Windows SmartScreen Warning

Because Null is not signed with a paid certificate, Windows may show a "Windows protected your PC" dialog when you run the installer.

This is expected. Click More info → Run anyway to proceed. The app is open source — you can verify the build yourself.
